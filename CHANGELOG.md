- 标签详情页: 改为滚动分页加载相关 VN

## 搜索改动

- 高级搜索面板完善:
  - 语言筛选 (11 种语言)
  - 平台筛选 (20 种平台)
  - 发行年份区间
  - 最低评分滑块
  - 标签 ID 筛选
  - 开发商筛选
  - 6 种排序方式 (搜索相关度/评分/投票数/发行日期/标题/ID)
  - 支持 compact filter 字符串粘贴
  - 重置按钮一键清空所有筛选
  - 动态构建 VNDB API filter 数组
  - **修复**: 评分筛选值类型修正为整数 (原先发送 `70.0` 导致 API 400 错误, 现发送 `70`)

### 2. 随机 VN 页面 

主页随机按钮从"直接抽取"改为进入**随机 VN 配置页面**, 支持以下筛选条件:

| 筛选项 | 说明 |
|--------|------|
| 语言 | 含有 / 不含有 (10种语言: 英语/日语/中文/韩语/法语/德语/西班牙语/俄语/意大利语/葡萄牙语) |
| 平台 | 18种平台 (Windows/Linux/macOS/Android/iOS/PS2/PSP/PSV/PS3/PS4/Switch/DS/3DS/Wii/WiiU/Xbox360/XboxOne/Web) |
| 标签 | 含有 / 不含有 (输入标签 ID, 如 g1) |
| 最低评分 | 滑块 0-100 |
| 发行年份 | 起止年份范围 |
| 开发商 | 输入开发商 ID, 如 p1 |
| R18 内容 | 全部 / 仅 R18 / 非 R18 (基于 image.sexual 字段客户端过滤) |
| 免费 | 全部 / 仅免费 / 非免费 (通过 /release 端点 freeware 过滤器预查询) |
| 引擎 | 输入引擎名称, 如 KiriKiri (通过 /release 端点 engine 过滤器预查询) |
