# VNDB Client v1.3.0 Changelog

## 版本

- 版本号升级至 **1.3.0** 1.2.0丢失(
- 替换全新应用图标 

## 首页改动

### I. 九宫格菜单

- 图标颜色随主题色变动 
- 九项菜单增加中文翻译, 中英双语显示


### II. 链接替换

### III. 最近更改

- "更多" 按钮不再跳转搜索, 改为打开 WebView 显示最近更改页面

### IV. 猜你喜欢 (新功能)

- 根据用户 wishlist 与 finished 列表中的 VN 标签进行推荐
- 聚合标签评分, 取 top 8 标签查询匹配 VN
- 排除用户已拥有的 VN
- 完全随机: 每次刷新结果不同 (标签顺序 + 结果池双重洗牌)
- 支持 "换一批" 按钮单独刷新推荐

### V. 即将发售 / 刚刚发售

- 卡片支持封面图展示 

### VI. Tags 浏览完善

- 标签列表页: 增加分类筛选 (全部/内容/色情/技术)、排序选项 (名称/数量)、滚动分页、下拉刷新
- 标签详情页: 改为滚动分页加载相关 VN

## 搜索改动

- 高级搜索面板完善:
  - 语言筛选 (11 种语言)
  - 平台筛选 (20 种平台)
  - 发行年份区间
  - 最低评分滑块
  - 标签 ID 筛选
  - 开发商筛选
  - 6 种排序方式 (搜索相关度/评分/投票数/发行日期/标题/ID)
  - 支持 compact filter 字符串粘贴
  - 重置按钮一键清空所有筛选
  - 动态构建 VNDB API filter 数组
  - **修复**: 评分筛选值类型修正为整数 (原先发送 `70.0` 导致 API 400 错误, 现发送 `70`)
